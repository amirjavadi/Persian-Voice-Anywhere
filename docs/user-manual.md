# راهنمای کاربر — Persian Voice Anywhere

نرم‌افزاری که در هر برنامه‌ی ویندوز فقط با صحبت‌کردن، متن فارسی را همان‌جا تایپ می‌کند —
کاملاً آفلاین.

## نصب و اجرا (پرتابل)

۱. فایل ZIP را در هر پوشه‌ای باز کنید (نیازی به نصب یا Administrator نیست).
۲. مدل‌ها را در پوشه‌ی `models/` کنار `Pva.App.exe` قرار دهید:
   - `silero_vad.onnx` (تشخیص گفتار)
   - `ggml-base.bin` یا مدل فارسیِ whisper.cpp
   (راهنمای دریافت: `docs/models.md`)
۳. `Pva.App.exe` را اجرا کنید. برنامه کنار ساعت ویندوز (System Tray) و به‌صورت یک
   میکروفون شناور روی صفحه ظاهر می‌شود. تنظیمات و لاگ کنار همان فایل ساخته می‌شوند.

## استفاده

۱. مکان‌نما را در هر برنامه‌ای (Word، Chrome، Telegram، Notepad و…) قرار دهید.
۲. کلید میانبر (پیش‌فرض **Ctrl+Space**) را نگه دارید و صحبت کنید؛ رها که کنید، متن
   فارسیِ دقیق همان‌جا تایپ می‌شود. یا روی گوی میکروفون شناور کلیک کنید.
۳. حالت‌ها روی میکروفون شناور نمایش داده می‌شود: «آماده»، «در حال شنیدن»،
   «در حال پردازش»، «در حال تایپ».

## دستورهای صوتی

این عبارات به‌جای نوشته‌شدن، به کنش تبدیل می‌شوند:

| بگویید | نتیجه |
|--------|-------|
| «خط بعد» / «خط جدید» | رفتن به خط بعد |
| «پاراگراف جدید» | پاراگراف تازه |
| «ویرگول» / «نقطه» / «علامت سوال» / «علامت تعجب» | ‪،‬ ‪.‬ ‪؟‬ ‪!‬ |
| «دو نقطه» / «نقطه ویرگول» | ‪:‬ ‪؛‬ |
| «پرانتز باز» / «پرانتز بسته» | ( ) |
| «گیومه باز» / «گیومه بسته» | « » |
| «حذف کلمه قبل» / «پاک کن» | حذف کلمه‌ی قبلی |
| «بازگردانی» (Undo) / «بازانجام» (Redo) | واگرد / ازنو |

می‌توانید تفسیر دستورها را در تنظیمات خاموش کنید تا همه‌چیز عیناً دیکته شود.

## کیفیت فارسی

متن به‌صورت خودکار اصلاح می‌شود: نیم‌فاصله (مثل «می‌روم»، «کتاب‌ها»)، علائم نگارشی،
اعداد فارسی، و حفظ اصطلاحات انگلیسی/فنی (مثل «امروز یک Pull Request روی GitHub زدم.»).

## نوت‌پد داخلی

از منوی Tray → «نوت‌پد داخلی». ویرایشگر تب‌دار با پشتیبانی راست‌به‌چپ، Word Wrap،
Undo/Redo و **بازیابی خودکار session** (تب‌های باز پس از بستن برنامه بازمی‌گردند).

## یادداشت‌های چسبان

از منوی Tray → «یادداشت چسبان جدید». یادداشت کوچک روی دسکتاپ که می‌توانید جابجا،
پین (روی همه‌ی پنجره‌ها) یا حذف کنید. موقعیت و محتوا خودکار ذخیره می‌شود.

## تنظیمات

از منوی Tray → «تنظیمات»:
- **کلید میانبر:** مثل `Ctrl+Space`، `CapsLock`، `DoubleCtrl`.
- **حالت:** Push-to-Talk (نگه‌داشتن) یا Toggle (روشن/خاموش).
- **موتور:** whisper.cpp (پیش‌فرض) یا Faster Whisper (در صورت نصب).
- **GPU:** استفاده در صورت وجود کارت گرافیک.
- **اصلاح هوشمند فارسی، ارقام فارسی، دستورهای صوتی، شفافیت میکروفون.**

## رفع اشکال

- **در یک برنامه تایپ نمی‌شود:** اگر آن برنامه با دسترسی Administrator اجرا شده،
  Persian Voice Anywhere را هم با دسترسی Administrator اجرا کنید.
- **آنتی‌ویروس هشدار می‌دهد:** برنامه از hook کیبورد و SendInput استفاده می‌کند (مثل
  ابزارهای مشابه)؛ در نسخه‌ی امضاشده این هشدار برطرف می‌شود.
- **«مدل یافت نشد»:** فایل‌های مدل را در `models/` قرار دهید (بالا را ببینید).

## حریم خصوصی

تمام پردازش **روی دستگاه شما و آفلاین** انجام می‌شود. هیچ صدا یا متنی به اینترنت
ارسال نمی‌شود.
